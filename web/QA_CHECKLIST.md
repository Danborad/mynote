# MyNote Web 最小回归清单

## 1. 认证与基础
- 打开 Web 首页，未登录时显示登录页。
- 使用正确账号登录成功并进入笔记列表。
- 登出后回到登录页。

## 2. 笔记编辑流程
- 新建笔记后进入编辑页，可正常输入并自动保存。
- 笔记列表排序正确：置顶在上，其余按更新时间倒序。
- 列表摘要保留换行语义（标题行与链接行不拼接）。
- 上传图片后可立即显示，切换到其他笔记再返回仍可见。
- 上传后若网络抖动，前端会自动重试一次保存（无需手动刷新）。

## 3. 分享功能
- 编辑页点击“链接分享”可生成并复制/调起分享。
- 分享链接 `/share/:token` 可匿名只读访问。
- 分享页可正常显示代码块（含复制按钮）、图片、音频、视频。
- 编辑页“复制分享链接”“关闭链接分享”动作可用。

## 4. 设置页
- 用户名修改弹窗：可打开、保存、关闭、ESC 关闭。
- 密码修改弹窗：校验生效（空值、长度、确认密码一致）。
- 废纸篓保留时间弹窗：7天/15天/1个月/3个月可切换并保存。
- 导入 `.md/.markdown/.txt` 后返回设置页且顶部栏正常。

## 5. 移动端交互
- 移动端首页底部仅保留 `+` 按钮。
- 侧边栏打开时 `+` 按钮隐藏。
- 编辑页顶部悬浮栏显示正常；“更多选项”菜单可点击。
- 移动端编辑页“更多选项”包含分组/分享/导出/管理分区，支持移动到分组。
- 笔记列表可滚动，底部过渡无明显硬切。

## 6. 首页布局偏好
- 设置 -> 偏好设置可切换首页卡片排布（单列/双列），切换后立即生效。
- 双列模式列表区域会加宽，卡片瀑布流无明显大空洞。
- 设置 -> 偏好设置可切换卡片密度（标准/紧凑/超紧凑），刷新后保持。

## 7. 侧边栏统计
- 上传附件后“已用空间”实时更新。
- 热力图按日期活动显示，hover 可看到“日期 + 条数”。

## 8. 性能与加载
- 首次加载无白屏错误。
- 关键页面采用懒加载后可正常切换（列表/设置/分享/编辑）。
- `html2canvas/turndown` 等重依赖按需加载，首次进入编辑无明显卡顿。
